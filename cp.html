1901A - Line Trip

Insight: calc the distance from start to the first station. then from the first to the last. then double the distance from the last station to destination X.
View Solution 

t = int(input()) # testcases
for _ in range(t):

    n_stations,x_dest= map(int, input().split()) # inputs
    station_indices = list(map(int, input().split()))

    gaps = [] # sol
    gaps.append(station_indices[0]-0)

    for i in range(len(station_indices)-1):
        gaps.append(station_indices[i+1]-station_indices[i])

    gaps.append(2*(x_dest-station_indices[-1]))

    print(max(gaps))
