1903A - Halloumi Boxes

Insight: if you can reverse a subarray of length 2 (kâ‰¥2), you are swapping two neighbors. if you can swap neighbors, you can do Bubble Sort. Bubble Sort can sort any array.

for _ in range(t):

    n,k = map(int, input().split()) # inputs
    a = list(map(int, input().split()))

1901A - Line Trip

Insight: calculate the distance from the start to the first station. then from the first to the last. then double the distance from the last station to destination X.
View Solution 

t = int(input()) # testcases
for _ in range(t):

    n_stations,x_dest= map(int, input().split()) # inputs
    station_indices = list(map(int, input().split()))

    gaps = [] # sol
    gaps.append(station_indices[0]-0)

    for i in range(len(station_indices)-1):
        gaps.append(station_indices[i+1]-station_indices[i])

    gaps.append(2*(x_dest-station_indices[-1]))

    print(max(gaps))

1890A - Doremy's Paint 3

Insight: b1+b2=b2+b3=b3+b4 simplifies to b1=b3 and b2=b4. it means that we can have at most 2 distinct values. also, the number of these values must be balanced (equal or off by 1).
View Solution 

from collections import Counter # import

t = int(input()) # testcases
for _ in range(t):

    n = int(input()) # inputs
    a = list(map(int, input().split()))

    counts = Counter(a) # dictionary (keys:values)
    freqs = list(counts.values()) # list with no keys (only appearances)

    if len(counts) == 1:
        print('yes')
    elif len(counts) == 2:
        if abs(freqs[0]-freqs[1]) <= 1:
            print('yes')
        else:
            print('no')
    else:
        print('no')
