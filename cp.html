Competitive Programming

1903A - Halloumi Boxes

Insight: If you can reverse a subarray of length 2 (meaning k≥2), you are effectively swapping two neighbors ([A,B]→[B,A]). If you can swap neighbors, you can perform Bubble Sort. Bubble Sort can sort any array arrangement.
View Solution 

t = int(input())

for _ in range(t):
    n,k = map(int, input().split())
    a = list(map(int, input().split()))

    if k >= 2 or a == sorted(a):
        print("YES")
    else:
        print("NO")
